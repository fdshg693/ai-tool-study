# フック機能の使い方
```.claude/settings.json``` に記述する

## 基本構造
```json
{
  "hooks": {
    "EventName": [
      {
        // ツール呼び出しのパターン
        "matcher": "ToolPattern",
        // パターンがマッチした場合に、実行するフック（複数指定可）
        "hooks": [
          {
            "type": "command",
            "command": "your-command-here"
          }
        ]
      }
    ]
  }
}
```

- matcher: ツール呼び出しのパターンを指定する。  
  - PreToolUse, PermissionRequest, PostToolUseのイベントでのみ使用可能。  
  - 正規表現をサポート。
    - Write -> Writeへの完全一致
    - Write.* -> Writeで始まる全てのツール呼び出し 
    - Edit|Write -> EditまたはWriteのツール呼び出し
    - * -> 全てのツール呼び出し
