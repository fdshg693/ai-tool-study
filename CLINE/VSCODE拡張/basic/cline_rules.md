# .clinerules/フォルダについて
- .clinerules/配下のマークダウンファイルは、全て自動的に１つのルールセットとしてCLINEに読み込まれます。
    - github copilotのinstructionsフォルダとほぼ同等です。
    - CLINEのチャット画面の下の＋ボタンを押して、ルールセットが管理できます。
- 便利な点
    - 簡単に、ルールを活性・非活性にクリックで切り替えられる点が、github-copilotにはない点です。
    - また、フォルダ直下でなくとも、配下にあれば読み込まれるので、サブフォルダで分類して管理できます。
