# Focus Chainについて
- 設定から、Focus ChainのON/OFFを切り替えることができます。
    - 機能としては、TODOリストを自動生成することで、進捗を管理しやすくするものです。
        - Github Copilotの場合は、Toolを使ってTODOリストを生成します。
        - 相違点
            - TODOリストをUSERが編集することが可能です
            - ファイルとして、生成されるのでGIT管理に含めたり、異なる会話に組み込んだりすることが可能です。
            - Remind Cline Intervalから、どれほどの頻度でTODOリストを更新するかを設定できます。
