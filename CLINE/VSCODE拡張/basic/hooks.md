# HOOKS
特定のイベントが発生した際に、自動的に実行されるスクリプトやコマンドを定義するための仕組みです。  
サブスクライブできるイベントは、CLINEが提供するもののみです。
- 注意点
    - Windowsでは使えず、Mac/Linuxのみで利用可能です。
- 設定
    - 設定より、hooksを有効化・無効化できます。
- サブスクライブできるイベント種類
    - PreToolUse: ツールが使用される前にフックが実行されます。
    - PostToolUse: ツールが使用された後にフックが実行されます。
    - UserPromptSubmit: ユーザープロンプトが送信された後にフックが実行されます。
    - TaskStart: タスクが開始される際にフックが実行されます。
    - TaskResume: 中断されたタスクが再開される際にフックが実行されます。
    - TaskCancel: タスクがキャンセルされる際にフックが実行されます。
- フックの定義方法
    - .clinerules/hooks配下に、上のイベントと同名のスクリプトを作成します。
    - chomod +x で実行権限を付与します。
