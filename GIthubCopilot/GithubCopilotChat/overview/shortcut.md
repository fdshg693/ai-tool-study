#　ショートカットまとめ

## Mac
- ⌃⌘I 
    - Open the Chat view
- ⌘I 
    - Enter voice chat prompt in Chat view
- ⌘N 
    - Start a new chat session in Chat view
- ⇧⌘I 
    - Switch to using agents in Chat view
- ⌘I 
    - Start inline chat in the editor or terminal
- ⌘I (hold) 
    - Start inline voice chat
- Tab 
    - Accept inline suggestion or navigate to the next edit suggestion
- Escape 
    - Dismiss inline suggestion
- ⇧⌥⌘L	
    - Open Quick Chat without interrupting your workflow.
