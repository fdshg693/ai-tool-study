# サブエージェントツール

- サブエージェントは、メインエージェントから渡されたプロンプト・ユーザが渡したコンテキストのみを使用してタスクを実行する、独立したエージェントです。
    - そのため、メインエージェントがその前に様々なファイルを読んでいたとしても、サブエージェントにはユーザが渡したコンテキスト以外の情報は渡されません。
- サブエージェントは、メインエージェントと同様に、ツールを使用できます。
- また、複数のサブエージェントを呼び出すことが可能です。ただし、その場合並列実行はサポートされておらず、順次実行されます。
- サブエージェントのモデルは、現状メインエージェントと同じモデルが使用されて、変更はサポートされていません。
